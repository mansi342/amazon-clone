# 🛒 Amazon Clone (Frontend)

## 📌 Overview

This is a frontend Amazon clone built using HTML, CSS, and JavaScript. It replicates the basic structure and design of an e-commerce website with product listings, cart page, and responsive layout.

## 🚀 Features

* Home page with product sections
* Shopping cart UI
* Login page UI
* Responsive design (mobile + desktop)
* Clean and user-friendly interface

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
